## About

This is my second creative project created for college. You'll have to answer questions in a limited amount of time.
It's an updated version of the old game version. Updated code, graphics, sounds and quiz system. 
Old version may be released later (not sure).

## How to convert quizes into an executable file

Requirements:
- `Python 3.11+`
- `PyInstaller`

Run the `test_to_exe.bat` to start the conversion.
A `NewTest` folder will be created, containing the converted quizes into the executable file.
